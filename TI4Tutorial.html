Poop dust.

Overview

  • Components
  • Objectives
    • Reach 10 victory points through Public Objectives and UP TO 3 Secret Objectives
    • Game ends when no more Public Objective cards can be revealed, in which case, highest VP wins
    • Objective cards will indicate during which phase they can be scored
      -- Only 1 Public and 1 Secret objective can be scored per phase or step
    • A player can only have 3 Secret Objectives TOTAL, scored or unscored
      -- If they draw a 4th one, they must return an unscored one back to the deck and shuffle
  • Command Pool
    • Represents your ability to issue commands each turn:
      • 3 Tactic tokens
            -- Spend to issue command to Activate a system (Tactical Action)
      • 3 Fleet tokens
            -- Indicated max fleet size in 1 system (Excluding Fighters and Infantry)
      • 2 Strategy tokens
            -- Spend to take part in Strategic Action
    • Tokens spent are removed
    • Receive 2 more per round (At Status Phase)
    • Can be redistributed (At Status Phase)
  • Round Structure
    • 1. Strategy Phase
    • 2. Action Phase
    • 3. Status Phase
    • 4. Agenda Phase (when unlocked)

Strategy Phase

  • Choose Strategy card for the round
  • Current Speaker chooses first
  • Number on Strategy card = Initiative Order = turn order
  • Strategy cards not chosen gain 1 Trade Good per turn (ie becomes more desirable)
  • Each Strategy provdes the player a powerful Primary Ability
        -- Other players can use Secondary Ability on the card if they spend Strategy command tokens

Action Phase

  • Go around in Initiative Order until everyone has passed
  • Choose 1 to execute each turn:
    • Activate a system (aka Tactical Action)
      Steps of Action Phase:
      • 1. Place a Tactic command token on system to Activate it
            -- Cannot activate if system already has one of your Tactic tokens
      • 2. May have units MOVE TO (not from) active system
      • 3. Space combat (if more than 1 player's ship in system)
      • 4. Invasion (send ground troops to planet)
      • 5. Production if system contains a Space Dock (limited by Fleet pool)
    • Use Strategy card (aka Strategic Action)
      • Use Strategy card's Primary Ability
      • OTHER players may use Secondary Ability (often costs Strategy command tokens)
      • Must use this at some point in the round before you're able to pass turn
      • Players who have passed may still take part in Secondary Ability
      • Flip card over after. It's no longer available this round
    • Use "Action:" ability on a component (aka Component Action)
      • Game components that say "Action:", ie Action Cards, Faction Sheets, Tech cards
  • Player cannot pass until they've at least completed their Strategic Action

-- Movement Rules

  • Ships MOVE TO Active System
  • Can call on any number of ships, but must maintain Fleet limit
  • Any ship may move to system as long as they have high enough Move value (1 = 1 hex)
  • May take ships from multiple systems
  • Cannot take ships from systems with a Command token (ie, previously activated)
  • Cannot move through enemy ship or Fighter occupied space
  • May transport Fighters and Infantry up to Capacity of ship
  • Can pick up but not drop-off along the way
  • Players in active system may use Space Cannon abilities
  • Wormholes are adjacent (a/a and b/b)
  • Asteroid Fields cannot be moved through or into
  • Supernova cannot be moved through or into
  • Nebula can be moved into, but not through. Defenders have +1 to combat. Ships move out at Move of 1
  • Gravity Rifts give +1 movement when moving through or out, but must roll D10 for each ship and destoy on 3 or less

-- Space Combat Rules

  • Anti-fighter Barrage
    • Can only destroy Fighters (essentially cannon fodder)
  • Announce Retreat
    • Either player can announce intention to retreat
    • Ground forces can be picked up for retreat (limited by Capacity)
  • Combat Rolls
    • Both players roll one D10 for each ship individually
    • If it's equal or higher than the ship's combat value, it's a HIT
    • Ships with Burst icons roll that many dice instead
  • Assign Hits
    • Players choose and destroy THEIR OWN SHIPS
    • 1 hit against = 1 ship destroyed, regardless of value
  • Retreat
    • Can only retreat if announced earlier
    • Move ALL ships to adjacent system they control (not simply unoccupied)
    • Cannot retreat if no system available
    • Place Command token from Reinforcements in destination system (if it doesn't have one already)
  • Combat ends when all units from either side is destroyed or retreated
  • Otherwise start next combat from Announce Retreat step
  • Player has won the combat if they have at least 1 ship remaining in Active System

-- Trading

  • Each turn active player can make 1 deal with each other player
  • Promissory Notes, Commodities, Trade Gods and favors can be traded
  • Can do so even mid combat
  • If components are involved (ie Commodities), players must be neighbours (share borders)
  • Simple (ie Commodities for Commodities) or immediate (Commodities for retreat) deals are BINDING DEALS
  • Delayed deals (ie goods for favor in next turn) are NON-BINDING and can be honoured or broken

-- Passing

  • Only available after Strategy card has been used
  • Can still participate in other players' Strategy cards after passing

Status Phase

  • Score up to 1 Public and 1 Secret objective
  • Next available Public Objective is revealed
  • Draw 1 Action card (max of 7, discard down if over after drawing)
  • Remove command tokens from board and return to Reinforcements
  • Gain 2 command tokens and redistribute Command Pool
  • Ready all exhausted cards (ie planets and tech)
  • Repair units
  • Return Strategy cards
  • Start Agenda phase if Mecatol Rex has been claimed, otherwise New Round (Status Phase)

Agenda Phase

  • Unlocked once Mecatol Rex has been claimed
  • Speaker draws first Agenda
  • Players starting from left of Speaker announce their votes (Speaker votes last)
  • Vote by exhausting any number of planets
  • 1 point of Influence = 1 vote For or Against the Agenda (or to elect a player/planet, depending on card)
  • Trade Goods cannot be used as Influence for votes
  • Players cannot vote for more than 1 option, but may abstain
  • Speaker breaks ties
  • Resolve Agenda
  • Speaker then draws a SECOND Agenda
  • Same procedure, except planets that already voted (exhausted) CANNOT vote again
  • Players may make deals during each vote to bribe other players
  • Components may be traded during this phase even if players are not neighbours
  • Ready all planets after both votes, then start New Round (Strategy Phase)
